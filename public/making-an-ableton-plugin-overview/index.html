Making an Ableton Plugin: Overview

18 Jul, 2021

Hi. New website new blogs.

I wanna make an Ableton plugin that rolls chords on the piano roll. These blogs will detail that process from absolute scratch. This overview page will be updated accordingly as things become fleshed out. No better way to start than to start, right? I know very little C++ and no DSP theory, so these will be pretty murky waters.
